# Display Image Name - Figma Plugin

## Short Description
Automatically create clean, readable labels showing the original filenames of your imported images, positioned neatly above each image.

## Detailed Description
Display Image Name is a time-saving plugin that helps you keep track of your image assets by automatically labeling them with their original filenames. Perfect for designers who work with multiple images and need to maintain organized, easily identifiable assets in their designs.

### Key Features
• Smart Name Detection: Automatically extracts original filenames from image metadata when available
• Clean Positioning: Places labels above images without overlapping the design
• Automatic Grouping: Labels are grouped with their images for easy movement and organization
• Batch Processing: Label multiple images at once with a single click
• Non-Destructive: Labels are added as separate text elements that can be easily modified or removed

### How It Works
1. Select one or more images in your design
2. Click "Display Image Name" for a single image or "Display Image Names" for multiple images
3. Labels appear above each image, showing their original names
4. Labels automatically move with their images, staying perfectly aligned

### Perfect For
• Design Systems with multiple image assets
• UI/UX design projects requiring organized image management
• Design handoff and documentation
• Team collaboration where image identification is crucial

### Technical Details
• Font: Uses system fonts for maximum compatibility
• Text Size: 14px for optimal readability
• Color: Pure black (#000000) for clarity
• Position: 8px gap above images
• Works in both Figma and FigJam

### Pro Tips
• Works with any shape that has an image fill
• Labels can be easily styled or modified after creation
• Compatible with both single images and batch processing
